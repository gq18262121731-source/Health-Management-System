# 知识库原始文档目录

## 📚 说明

这个目录用于存放**原始的**知识库文档（上传前的文档文件）。

## 📁 目录结构

```
knowledge_base/
├── docs/              # 原始文档目录（本目录）
│   ├── health/        # 健康知识文档
│   ├── medication/    # 用药指南
│   └── faq/          # 常见问题
├── vectors/          # 向量数据库（自动生成）
├── documents/        # 索引后的文档（自动生成）
└── README.md         # 说明文件
```

## 📝 使用方法

### 方式1：直接上传文件（推荐）

将文档放在这个目录中，然后通过API上传：

```bash
# 上传文档
curl -X POST "http://localhost:8000/api/knowledge-base/upload" \
  -F "file=@health/高血压管理指南.txt" \
  -F "title=高血压管理指南" \
  -F "source=医疗健康指南"
```

### 方式2：使用批量导入脚本

可以创建一个脚本，批量导入本目录下的所有文档。

## 📄 支持的文件格式

- `.txt` - 纯文本文件
- `.md` - Markdown文件（推荐）
- `.pdf` - PDF文档
- `.docx` - Word文档

## 💡 建议

1. **文档命名**：使用有意义的文件名，如 `高血压管理指南.md`
2. **内容组织**：使用清晰的标题和章节结构
3. **分类存放**：可以创建子目录按主题分类
4. **定期更新**：保持文档内容的最新性

## 🚀 快速开始

1. 将你的健康知识文档复制到这个目录
2. 按照文档类型创建子目录（可选）
3. 使用API上传文档到知识库
4. 文档会自动索引，供AI助手使用
