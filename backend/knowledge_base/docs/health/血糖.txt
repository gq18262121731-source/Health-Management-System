# 血糖控制要点
1. **基础控制目标**
   糖化血红蛋白（HbA1c）<7%；空腹血糖4.4~7.0 mmol/L；餐后2小时血糖（或高峰值血糖）<10.0 mmol/L（证据级别I，推荐级别A）。

2. **特殊人群目标放宽**
   老年、易发生低血糖、病程长、合并症/并发症多、预期寿命短、医疗资源不足或无条件监测血糖的患者，血糖控制目标可适当放宽（证据级别IIa，推荐级别B）。

3. **药物选择原则**
   - 大多数2型糖尿病患者，首选二甲双胍；
   - 无论HbA1c是否达标：
     - 合并动脉粥样硬化性心血管疾病（ASCVD）或心血管高危风险的患者，需联用SGLT2i（钠-葡萄糖协同转运蛋白2抑制剂）或GLP-1RA（胰高糖素样肽-1受体激动剂）；
     - 合并慢性肾脏病（CKD）或心力衰竭的患者，需联用SGLT2i（证据级别I，推荐级别A）。

# 高血压患者的血糖控制
1. 控制原则
通过健康生活方式+药物，综合控制多种代谢性心血管危险因素

2. 基础血糖控制目标
- 糖化血红蛋白（HbA1c）：<7%
- 空腹血糖：4.4~7.0 mmol/L
- 餐后2小时血糖/非空腹血糖：<10.0 mmol/L

3. 目标可放宽的人群
- 老年患者
- 易发生低血糖的患者
- 糖尿病病程长的患者
- 合并症/并发症多的患者
- 预期寿命较短的患者
- 医疗资源差、无条件监测血糖的患者
- 1型糖尿病合并肾脏病、严重眼底病等并发症的患者

4. 放宽的基本原则
不发生低血糖和高血糖急症

# 糖尿病综合管理原则
## 一、饮食调整原则
1. 控制总热卡，碳水化合物占总热量55%~65%，蛋白质占比不超过总热量15%；
2. 尽可能将体重控制在正常范围内；
3. 总热量不变前提下，采取少食多餐模式，利于血糖控制并避免低血糖。

## 二、运动和活动原则
1. 遵循适量、经常性、个体化原则；
2. 推荐步行、游泳等骨骼肌等张运动，避免剧烈运动及过量运动；
3. 接受胰岛素治疗的患者，需保持规律生活，做到定时定量进餐和运动。

## 三、药物治疗主要原则[336-338]
1. 大多数2型糖尿病患者首选二甲双胍；体重偏瘦或单用二甲双胍血糖控制不佳者，改用或加用其他降糖药；
2. HbA1c水平高于治疗靶目标1.5%~2.0%的患者，初始即考虑联合降糖药治疗；
3. 并发症和合并症是降糖药选择的重要依据：
   - 合并ASCVD或心血管高危风险者，无禁忌证时，无论HbA1c是否达标，均使用具有心血管获益证据的GLP-1RA或SGLT2i；
   - 合并CKD或心力衰竭者，无禁忌证时，无论HbA1c是否达标，均使用SGLT2i；
4. 合并肾脏疾病的患者：
   - eGFR≥30 mL/(min·1.73 m²)时，使用SGLT2i，降低糖尿病肾病进展和/或心血管事件风险；
   - eGFR≥30 mL/(min·1.73 m²)时，可考虑使用GLP-1RA，减少新发大量蛋白尿风险；
   - 肾功能不全者优先选择肾脏排泄较少的降糖药，严重肾功能不全者宜采用胰岛素治疗；
5. 2种中等以上剂量降糖药仍难以控制血糖时：
   - 可采用口服降糖药+每日1次基础胰岛素（中效、超长效胰岛素或其类似物）治疗；
   - 或先采用口服降糖药+GLP-1RA治疗（每周1次周制剂患者接受度更高）；
   - 仍控制不佳者，采用每日多次胰岛素注射治疗；
6. 新诊断2型糖尿病患者，若空腹血糖>11 mmol/L或HbA1c≥9%，且伴明显高血糖症状，可考虑短期（2周~3月）胰岛素强化治疗，快速控制高血糖、保留胰岛细胞功能；
7. 1型糖尿病患者，需在规律生活方式和自我血糖监测基础上，依赖胰岛素治疗；
8. 降压治疗中需关注药物对血糖的影响：
   - 大剂量长时间应用噻嗪类利尿剂可能导致血糖升高；
   - β受体阻滞剂可能掩盖低血糖引发的心率增快等反应。
