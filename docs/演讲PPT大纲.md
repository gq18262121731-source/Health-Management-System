### 第7页：IQR异常值检测
```
[左侧：箱线图]
    ×  ← 异常值（200）
    ┃
  ┌─┃─┐
  │ ┃ │  ← 正常范围
  └─┃─┘
    ┃

[右侧：公式和案例]
Q1 = 25%分位数 = 122
Q3 = 75%分位数 = 130
IQR = Q3 - Q1 = 8
上界 = Q3 + 1.5×IQR = 142

案例：张大爷血压突然200
→ 超过上界，判定为异常
→ 自动剔除，避免误判
```

**演讲词**：
"IQR方法基于四分位数，对极端值不敏感。比如张大爷血压突然飙到200，很可能是测量错误，系统会自动识别并剔除。"

### 第8页：Z-score标准化
```
[中心：正态分布曲线]

        μ-3σ  μ-2σ  μ  μ+2σ  μ+3σ
          ↓    ↓    ↓    ↓    ↓
        ┌─────────────────────┐
        │      ╱───╲          │
        │    ╱       ╲        │
        │  ╱           ╲      │
        │╱               ╲    │
        └─────────────────────┘
        99.7%的数据在±3σ内

[底部：应用]
z = (x - μ) / σ
|z| > 3 → 极端异常
```

**演讲词**：
"Z-score衡量数据偏离均值的程度。根据3σ原则，99.7%的正常数据应在±3σ内，超出则为极端异常。"

### 第10页：线性插值平滑 - 解决方案
```
[平滑评分曲线]

得分
  30 ┤████
     │    ╲
  20 ┤      ╲████
     │           ╲
  10 ┤             ╲████
     │                  ╲
   0 ┤                    ╲
     └─────────────────────────
      115  135  155  185 血压(mmHg)

[对比效果]
传统硬切割：
• 血压139 → 20分
• 血压140 → 10分（跳变10分！）

线性插值平滑：
• 血压139 → 17.5分
• 血压140 → 17.0分（只差0.5分）
```

**演讲词**：
"我们借鉴模糊逻辑思想，使用线性插值实现平滑过渡。血压139和140只差0.5分，而不是跳变10分，这更符合医学实际。"

### 第12页：Isolation Forest原理
```
[树形图对比]

正常点（需要多次分割）：
     根
    ╱  ╲
   ╱    ╲
  ╱      ╲
 ●正常    ...
 (深度=8)

异常点（快速被孤立）：
  根
 ╱  ╲
●异常 ...
(深度=2)

核心思想：
异常点更容易被"孤立"
```

**演讲词**：
"Isolation Forest的核心思想是：异常点更容易被孤立。正常点需要多次分割才能隔离，异常点只需少数几次。"

### 第15页：TOPSIS多准则决策
```
[坐标图]

        严重性
          ↑
          │  A+（理想解）
          │   ●
          │  ╱│╲
          │ ╱ │ ╲
    高血压● │  │
          │╲│  │
    运动不足●│  │
          │  ●睡眠不足
          │   A-（负理想解）
          └──────────→ 紧迫性

高血压：0.85（最优先）
运动不足：0.62

```

**演讲词**：
"TOPSIS找到最接近理想解、最远离负理想解的方案。高血压既严重又紧迫，相对接近度最高，优先级最高。"
