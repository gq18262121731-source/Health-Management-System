# 算法速查卡 - 演讲快速参考

## 🎯 核心算法一句话总结

| 算法 | 一句话总结 | 演讲比喻 |
|------|-----------|---------|
| **IQR** | 基于四分位数找异常 | 找出"离群的羊" |
| **Z-score** | 衡量偏离均值的程度 | 看你离"平均水平"有多远 |
| **模糊逻辑** | 处理边界模糊问题 | 不是非黑即白，而是有灰度 |
| **Isolation Forest** | 异常点容易被孤立 | 异常的人更容易被"孤立" |
| **AHP** | 科学确定权重 | 给不同因素"称重" |
| **TOPSIS** | 找最优方案 | 既要接近好的，又要远离坏的 |

## 📊 算法公式速记

### IQR异常值检测
```
Q1 = 25%分位数
Q3 = 75%分位数
IQR = Q3 - Q1
上界 = Q3 + 1.5 × IQR
下界 = Q1 - 1.5 × IQR
```
**记忆口诀**：一五倍IQR，上下加减分

---

### Z-score标准化
```
z = (x - μ) / σ
|z| > 3 → 极端异常
|z| > 2 → 可疑异常
```
**记忆口诀**：三西格玛原则，99.7%在范围内

---

### 线性插值平滑（借鉴模糊逻辑思想）
```
if x < 阈值1:
    得分 = 满分
elif x < 阈值2:
    得分 = 满分 - (x - 阈值1) / (阈值2 - 阈值1) × 分差
else:
    得分 = 低分
```
**记忆口诀**：线性过渡，平滑边界，无跳变

---

### 控制质量评分
```
总分 = 达标率×40% + 稳定性×30% + 水平×30%
```
**记忆口诀**：四三三，达标稳定加水平

---

### AHP权重计算
```
1. 构建成对比较矩阵
2. 计算特征向量
3. 归一化得权重
4. 一致性检验 CR < 0.1
```
**记忆口诀**：比较-特征-归一-检验

---

### TOPSIS排序
```
1. 归一化决策矩阵
2. 加权
3. 确定理想解 A+, A-
4. 计算距离 d+, d-
5. 相对接近度 C = d- / (d+ + d-)
```
**记忆口诀**：归一加权定理想，距离接近度排序

## 💡 常见问题快速应答

### Q1: 为什么用IQR而不是均值±标准差？
**A**: 医学数据常有偏态分布，IQR对极端值不敏感，更鲁棒。

### Q2: 模糊逻辑和传统阈值有什么区别？
**A**: 传统阈值是硬切割（139正常，140异常），模糊逻辑是平滑过渡（135是70%正常+30%偏高）。

### Q3: Isolation Forest为什么能检测异常？
**A**: 异常点特征与众不同，在随机分割时更容易被快速孤立。

### Q4: AHP的一致性检验是什么意思？
**A**: 检查专家判断是否自相矛盾。比如A>B, B>C, 但C>A就不一致。

### Q5: TOPSIS如何排序风险因素？
**A**: 计算每个风险因素到"理想最优解"和"理想最劣解"的距离，选择最接近最优、最远离最劣的。

### Q6: 为什么疾病权重是45%？
**A**: 通过AHP方法，结合专家意见和医学指南，科学确定的权重。
