# 智慧健康管理系统（养生之道）

## 项目简介

面向老年人的智慧健康管理系统，集成健康数据监测、AI健康助手、语音交互等功能。

## GitHub 仓库

🔗 **仓库地址**: https://github.com/gq18262121731-source/Health-Management-System.git

## 技术栈

### 前端
- React + TypeScript + Vite
- TailwindCSS + shadcn/ui
- Recharts（图表可视化）

### 后端
- FastAPI + SQLAlchemy
- PostgreSQL
- WebSocket（实时通信）

### AI & 语音
- DeepSeek API（AI对话）
- Edge-TTS（语音合成）
- Web Speech API（语音识别）

## 快速开始

### 后端启动
```bash
cd backend
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 访问地址
- 前端: http://localhost:3000
- 后端: http://localhost:8000

## 主要功能

- 📊 **健康数据监测** - 血压、血糖、心率、体温、睡眠等
- 🤖 **AI健康助手** - 智能问诊、健康建议
- 🔊 **语音交互** - 语音播报、语音输入
- 📈 **可视化报告** - 健康趋势图表、雷达图评估
- 👨‍👩‍👧 **多端支持** - 老人端、子女端、社区端
